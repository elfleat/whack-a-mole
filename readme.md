# Whack-a-Mole 

This is a HTML + CSS + JS game written without any libraries or frameworks.

## Want to try it out?
https://hack-a-mole.herokuapp.com/

## Want to run locally?
```
1- git clone https://github.com/elfleat/whack-a-mole.git
2- npm i
3- npm start
```

## Want to develop?
```
1- git clone https://github.com/elfleat/whack-a-mole.git
2- npm i
3- npm i -g browser-sync
3- npm run dev
```
